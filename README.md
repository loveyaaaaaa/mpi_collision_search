ТЕМА: Разработка параллельной MPI-программы поиска коллизий хеш-функции MD5/SHA1/STREEBOG для заданного множества ключей.

## Описание
Данная программа реализует параллельный алгоритм перебора (brute-force) для нахождения прообраза хеш-функции. Используется технология MPI (Message Passing Interface) для распределения нагрузки между вычислительными узлами.

Поддерживаемые алгоритмы:
- **MD5**
- **SHA1**
- **Streebog**

## Требования
Для сборки и запуска проекта необходимы:
* Компилятор C (gcc/clang)
* Реализация MPI (OpenMPI или MPICH)
* Библиотека OpenSSL (`libssl-dev`)
